\begin{proof}[提示]平方和的数字和，直到最后是一位数。
  \begin{align*}
    &1^2+2^2+3^2 = 14, && \implies 1 + 4 = 5\\
    &4^2+5^2+6^2 = 74, && \implies 7 + 4 = 14, && \implies 1+4=5\\
    &7^2+8^2+9^2 = 194,&& \implies 1+9+4=14, && \implies 1+4=5
  \end{align*}
  按这个规律，?号处填200，最后会得到5。
\end{proof}
